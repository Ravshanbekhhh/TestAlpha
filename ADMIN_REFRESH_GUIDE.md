# Admin Panel Yangilash Ko'rsatmalari

## Muammo: Eski interface ko'rinmoqda

Fayl o'zgargan, lekin browser eski versiyani cache dan ko'rsatmoqda.

## Yechim: Hard Refresh

### Chrome/Edge:
1. Admin panelni oching: http://localhost:8000/static/admin/
2. **Ctrl + Shift + R** bosing (yoki F12 -> Network -> "Disable cache" checkboxni belgilang)

### Agar ishlamasa:

**1. Browser Cache ni Tozalash:**
- F12 ni bosing (Developer Tools)
- Network tab ni oching
- "Disable cache" ni belgilang
- Sahifani yangilang (F5)

**2. Yoki Incognito Mode:**
- Ctrl + Shift + N (yangi incognito oyna)
- http://localhost:8000/static/admin/ ni oching
- Login qiling

## Natija:

Yangilangandan keyin ko'rishingiz kerak:
- MCQ Answers (1-35)
- Q33, Q34, Q35 da **A, B, C, D, E, F** (6 ta variant)
- **Open-Ended Questions (36-37)** section pastda
- Har bir savol uchun a) va b) input box

## Tekshirish:

Modal oynada pastga scroll qiling - "Open-Ended Questions (36-37)" bo'limini ko'rishingiz kerak.
